# Sales-Prediction-Using-Machine-Learning
Machine Learning project that predicts product sales using advertising data and regression models in Python. Includes data analysis, visualization, and model evaluation.
# Sales Prediction Using Machine Learning in Python

## Project Overview

This project predicts product sales based on advertising budgets using Machine Learning techniques in Python. The model analyzes how advertising spending on TV, Radio, and Newspaper affects product sales.

The goal is to build a regression model that accurately predicts future sales based on marketing investment.

---

## Dataset

The dataset contains advertising spending across different marketing channels:

Features:
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget

Target Variable:
- Sales

---

## Results

The regression model successfully predicts product sales based on advertising investment with strong correlation between marketing spend and sales performance.
